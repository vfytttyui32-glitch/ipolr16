# Лабораторная работа 16 - Интернет-магазин украшений

## Описание проекта
Данный проект представляет собой веб-приложение на Django - интернет-магазин украшений. 
Сайт предоставляет базовую информацию о магазине и его создателе.

- **[Об авторе](/about-author/)** - информация о создателе лабораторной работы
